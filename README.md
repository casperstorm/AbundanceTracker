# AbundanceTracker

A World of Warcraft addon that tracks Abundance stacks from your active Rejuvenations.

<img src="icon.png" alt="icon" width="128">

## Features

- Tracks your active Rejuvenations on yourself, party members, and raid members
- Caps at 12 tracked stacks for the full 96% Abundance bonus
- Uses a segmented bar with a highlighted current stack marker
- Supports dragging, locking, scale, width, and height adjustments

## Usage

Type `/abt` or `/abundance` to open the configuration window.

- `/abt lock` locks the bar
- `/abt unlock` unlocks the bar for dragging
- `/abt reset` resets the bar position

## Installation

1. Download the latest release
2. Extract the `AbundanceTracker` folder to your `Interface/AddOns` directory
